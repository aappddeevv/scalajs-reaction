Importing the object's contents brings a few implicit type converters into scope so you can use some "type inference driving" functions to create your style parts.

  10.  type IStyleSet = Dictionary[IStyle]

    Keys are usually logical names of your component, e.g. root, header, footer. Fabric also desginates subComponentStyles as a property that is an object with string->istyle mappings or a function that takes properties (an object) and returns string->istyle mappings.
